# 🌲 Vostok-Help | Интерактивная база знаний

[🔗 ОТКРЫТЬ БАЗУ ЗНАНИЙ](https://deasmean.github.io/Vostok-help/)

Удобный инструмент для выживших в **Road to Vostok**, позволяющий отслеживать прогресс выполнения квестов и быстро находить ключевые локации.

---

### 🛠 Основные функции:

*   ✅ **Трекер квестов:** Отмечайте выполненные задания одной кнопкой. Состояние сохраняется автоматически.
*   📝 **Интерактивный чек-лист предметов:** Кликайте на нужные предметы в списке «Требуется», чтобы зачеркнуть их, когда они найдены.
*   📊 **Динамическая статистика:** Наглядный счетчик общего количества квестов, выполненных и оставшихся заданий.
*   💾 **Local Storage:** Ваш прогресс не сбросится при обновлении страницы или закрытии браузера.


*Проект находится в стадии активной разработки. Данные обновляются по мере исследования игры.*
