### 🛠 Основные функции:
*   ✅ **Quest Tracker:** Сохранение прогресса выполнения заданий
*   📊 **Live Stats:** Счетчик активных и выполненных квестов.
*   🕵️ **Traders Info:** Подробная информация о расположении Снабженца и Доктора с фото-подсказками.
*   ⚡ **Filters:** Быстрая сортировка заданий по сложности (Easy, Intermediate, Hard).



*Проект находится в стадии активной разработки. Данные обновляются по мере выхода патчей.*
